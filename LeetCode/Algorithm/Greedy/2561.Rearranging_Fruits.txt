Q:) You have two fruit baskets containing n fruits each. You are given two 0-indexed integer arrays basket1 and basket2 representing the cost of fruit in each basket. You want to make both baskets equal. To do so, you can use the following operation as many times as you want:

Choose two indices i and j, and swap the ith fruit of basket1 with the jth fruit of basket2.
The cost of the swap is min(basket1[i], basket2[j]).
Two baskets are considered equal if sorting them according to the fruit cost makes them exactly the same baskets.

Return the minimum cost to make both the baskets equal or -1 if impossible.



Topics: Greedy, HashTable, Sorting

class Solution {
    public long minCost(int[] basket1, int[] basket2) {
        Map<Integer,Integer> map = new HashMap<>();
        for(int item : basket1){
        map.put(item,map.getOrDefault(item,0)+1);
        }
        for(int item : basket2){
        map.put(item,map.getOrDefault(item,0)-1);
        }

        ArrayList<Integer> track = new ArrayList<>();
        int minEle = Integer.MAX_VALUE;
        for(int key : map.keySet()){
            int cnt = Math.abs(0-map.get(key));
            if(cnt % 2 == 1) return -1;
            for(int i = 0 ; i < cnt/2 ; i++){
               track.add(key);
            }
            minEle = Math.min(minEle,key);
        }
        long cost = 0;
        Collections.sort(track);
        int n = track.size();
        // for(int i = 0 ; i < n ; i++)System.out.println(track.get(i));
        for(int i = 0 ; i < n/2 ; i++){
            // System.out.println(track.get(i));
            cost += Math.min(track.get(i),(long) 2*minEle); // use current smaller element or smallest element twice
        }
        return cost;
    }
}

Time Complexity: O(nlogn);
Auxiliary Space: O(n);
